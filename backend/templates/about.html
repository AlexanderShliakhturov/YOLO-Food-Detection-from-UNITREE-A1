О проекте

Данный проект - приложение обрабатывающее bag файлы с робособаки и распознающее на них еду

Стек технологий

Веб фреймворк fastapi послужил основой нашей backend разработки

Связующим звеном всех микросервисов выступила kafka

Для хранения файлов и их дальнейшего стриминга использовалось s3 хранилище

Для хранения данных использовалась база данных postgresql

Обработка видео осуществлялась при помощи onen cv

В качестве реализованной нейросети выступила yolov2
